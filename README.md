# 🚀 EngiPath-AI

نظام خبير ذكي يعمل بمثابة مستشار تقني لمواءمة مهارات المبتكرين الشباب مع المشاريع الهندسية والتقنية الأمثل لشغفهم وقدراتهم.

## 🌟 نبذة عن المشروع (About the Project)
منصة ذكية تهدف إلى حل مشكلة "تشتت الأفكار" و"حيرة البدايات" لدى الطلاب والمبتكرين (من سن 8 إلى 25 عاماً). يعمل النظام كمستشار تقني ذكي يقوم بتحليل مهارات المستخدم، اهتماماته، والمواد المتوفرة لديه، ليقترح له المسار الهندسي أو المشروع التطبيقي الأكثر ملاءمة لقدراته وشغفه.

## ✨ المميزات الرئيسية (Key Features)
* مواءمة دقيقة بين مهارات المستخدم والمشاريع التقنية.
* الاعتماد على قواعد بيانات هندسية مصنفة وخوارزميات مواءمة ذكية.
* تقديم خطة عمل واضحة للمشروع المقترح تشمل الأدوات والخطوات.
* واجهة تفاعلية سهلة الاستخدام مبنية بتقنيات حديثة.

## 🛠 التقنيات المستخدمة (Tech Stack)
* Programming Language: Python 🐍
* Web Framework: Streamlit
* Deployment: Streamlit Cloud / GitHub
* Logic: Expert System (Rule-based Matching)

## 📁 هيكل الملفات (Project Structure)
* app.py: الملف الأساسي لتشغيل التطبيق.
* requirements.txt: يحتوي على المكتبات البرمجية اللازمة.

  
تم تطوير هذا المشروع لتمكين الجيل القادم من المبتكرين ودعم التحول الرقمي في التعليم التقني.
