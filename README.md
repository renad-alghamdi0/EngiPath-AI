# 🚀 EngiPath-AI | مسار المهندس

نظام خبير ذكي يعمل بمثابة مستشار تقني لمواءمة مهارات المبتكرين الشباب مع المشاريع الهندسية والتقنية الأمثل لشغفهم وقدراتهم.

## 🌟 نبذة عن المشروع (About the Project)
منصة ذكية تهدف إلى حل مشكلة "تشتت الأفكار" و"حيرة البدايات" لدى الطلاب والمبتكرين (من سن 8 إلى 25 عاماً). يعمل النظام كمستشار تقني ذكي يقوم بتحليل مهارات المستخدم، اهتماماته، والمواد المتوفرة لديه، ليقترح له المسار الهندسي أو المشروع التطبيقي الأكثر ملاءمة لشغفه.

## ✨ المميزات الرئيسية (Key Features)
* مواءمة دقيقة: ربط مهارات المستخدم بمشاريع هندسية واقعية.
* محرك استدلال (Expert System): يعتمد على قواعد بيانات هندسية مصنفة وخوارزميات مواءمة ذكية.
* خريطة طريق واضحة: تقديم خطة عمل تشمل الأدوات، المهارات المكتسبة، وأول خطوة للتنفيذ.
* واجهة تفاعلية: تجربة مستخدم عصرية تعتمد على تقنيات الويب الحديثة (Glassmorphism).

## 🛠 التقنيات المستخدمة (Tech Stack)
* Logic Engine: JavaScript (Rule-based expert system logic).
* Reference Model: Python (Core logic documentation).
* Frontend: HTML5 & CSS3 (Modern & Responsive UI).
* Deployment: GitHub Pages.

## 📁 هيكل الملفات (Project Structure)
* index.html: الهيكل الأساسي للواجهة التفاعلية.
* style.css: ملف التنسيق الجمالي وتصميم الـ UI.
* script.js: المحرك الاستدلالي الذي يدير منطق الاقتراحات والمواءمة.
* model.py: التوثيق البرمجي لمنطق النظام الخبير بلغة Python.
