## 🤖 عن المشروع
نظام خبير ذكي يساعد المبتكرين والمستكشفين في تحديد مسارهم الهندسي الأمثل. يقوم النظام بتحليل اهتمامات المستخدم وما يتوفر لديه من أدوات ليقترح له أفضل مشروع أو تخصص تقني يناسب شغفه.

تم تصميم هذا النظام ليكون بوصلة لكل من يجد حيرة في بداية طريقه التقني، سواء كان يبحث عن مشروع "ميكانيكا"، "إلكترونيات"، أو "برمجيات".

## 🎯 مميزات النظام
* توجيه ذكي: يقترح عليك المسار بناءً على ميولك الشخصية.
* دعم القرار: يقلل من تشتت الأفكار ويضعك على أول الطريق.
* واجهة تفاعلية: تجربة مستخدم سهلة وسريعة للوصول للنتيجة.

## 🛠 التقنيات المستخدمة
* Python (model.py): المنطق البرمجي وقواعد البيانات الاستشارية.
* HTML & CSS: واجهة الموقع وتنسيق الألوان.
* JavaScript (script.js): ربط الأسئلة بالنتائج والتفاعلية.
* GitHub Pages: استضافة الموقع ونشره.
